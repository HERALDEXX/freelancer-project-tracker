Freelancer Project Tracker

A MySQL-based system to manage freelance clients, projects, and payment records — clean, efficient, and real-world ready.

What it does

  • 📇 Stores client contact and company info
  • 📁 Tracks projects per client with deadlines and budgets
  • 💰 Logs payment records and methods
  • 📊 Supports useful queries like total payments, project count, etc.
  • 🔌 Easy to import and run in any MySQL setup
